# Comandas para compilar e depois executar os codigos

## Compilar

```
gcc -c <nome do arquivo>.c
```

## Criar o .exe

```
gcc <nome do arquivo>.o -c <nome do arquivo>.exe
```

## Rodar o .exe

```
./<nome do arquivo>.exe
```

## Criar arquivo relatorio

```
gcc -E <nome do arquivo>.c -o <nome do arquivo>.i
```
